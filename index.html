Privacy Policy

Effective date: 10 March 2026  ·  Published by Nuno Lopes

ProfileFitCheck does not collect, store, transmit, or sell any personal data. All processing happens locally in your browser or via direct API calls you control.

1. What This Extension Does

ProfileFitCheck is a Chrome extension that helps recruiters evaluate candidate profiles on LinkedIn. It reads the visible text of the LinkedIn profile you are currently viewing, sends that text to Google's Gemini API (using your own API key), and displays the AI-generated analysis in the extension side panel.

2. Data We Collect

We do not collect any data. Specifically:

  • No personal data is collected, stored, or transmitted to any server operated by ProfileFitCheck or Nuno Lopes.
  • No analytics, telemetry, crash reports, or usage statistics are gathered.
  • No account or registration is required.

3. Data Processed Locally

The following data is processed entirely within your browser and is never sent to us:

  • Your Gemini API key — stored locally using Chrome's chrome.storage.local API. It never leaves your device except in direct API calls to Google's Gemini API, which you initiate.
  • Role / ICP profiles you create — stored locally in chrome.storage.local. Never transmitted anywhere.
  • Candidate scoring history and notes — stored locally in chrome.storage.local. Never transmitted anywhere.

4. Data Sent to Third Parties

When you click "Score Full Profile" or "Scan Companies", the extension sends the visible text of the LinkedIn profile you are viewing — along with your role ICP criteria — to the Google Gemini API using your own API key. This is a direct call from your browser to Google's API.

This data transfer is:

  • Initiated only when you explicitly trigger an analysis action.
  • Governed by Google's Privacy Policy and the Gemini API Terms of Service.
  • Subject to the data retention policies of Google, not ProfileFitCheck.

We recommend reviewing Google's API data usage policies before use, particularly regarding whether API inputs are used for model training.

5. LinkedIn Data

The extension reads the visible text content of LinkedIn profile pages you navigate to. This data is used solely to generate the analysis you requested. It is not stored beyond your local browser session history (which you can clear at any time) and is not transmitted to any party other than the Google Gemini API as described above.

Use of this extension in relation to LinkedIn data is your responsibility. Please ensure your use complies with LinkedIn's User Agreement and applicable data protection laws including GDPR.

6. Permissions Used

The extension requests the following Chrome permissions, each with a specific and limited purpose:

  • activeTab — to read the visible text of the LinkedIn profile tab you are currently viewing.
  • scripting — to inject a content script that extracts profile text from the page.
  • storage — to save your API key, role profiles, scoring history, and notes locally on your device.
  • sidePanel — to display the extension interface in Chrome's side panel.
  • tabs — to detect when you navigate to a LinkedIn profile page so the extension can update automatically.

7. Children's Privacy

This extension is intended for professional use by recruiters and talent acquisition professionals. It is not directed at children under the age of 13, and we do not knowingly process data relating to children.

8. Changes to This Policy

If this policy is updated, the effective date at the top of this page will be revised. Continued use of the extension after any changes constitutes acceptance of the updated policy.
